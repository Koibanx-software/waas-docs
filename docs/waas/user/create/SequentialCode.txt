title Flow to create a WAAS User

Client->WAAS:POST /auth/users 
note over Client,WAAS:(id,username)
WAAS->DB:The user exists in the project
DB->WAAS:The user does not exists
WAAS->DB: Save user
DB->WAAS: User Saved
WAAS->Client:Successfully Created User
WAAS->Ledger: Create Wallet
Ledger->WAAS: Wallet Created
WAAS->DB: Associate Wallet to user
DB->WAAS: Wallet Associated
