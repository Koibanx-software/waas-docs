title Flow to create a WAAS project

Client->WAAS:POST /projects 
note over Client,WAAS:(name,description)
WAAS->Build-Platform:Create Project In WAAS Organization
Build-Platform->WAAS:Project Created
WAAS->DB: Save Project
DB->WAAS: Project Saved
WAAS->DB: Associate Project to user
DB->WAAS: Project Associated
WAAS->Build-Platform:hire Modules
Build-Platform->WAAS:Modules hired
WAAS->Client:Successfully Created Project
